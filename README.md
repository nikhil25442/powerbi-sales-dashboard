# 📊 Sales Dashboard - Power BI

## 📌 Project Overview
This project showcases an interactive **Sales Dashboard** built using Power BI to analyze business performance and generate actionable insights.  
The dashboard helps in tracking sales, orders, product performance, and overall business growth.

---

## 🎯 Objectives
- Analyze total sales and order volume  
- Identify top-performing products and categories  
- Track yearly sales trends  
- Compare product line performance  

---

## 📊 Dashboard Features
- 📈 **Sales KPI**: ₹735K total sales with 7,959 orders  
- 🥇 **Top Products Analysis**: Identify highest revenue-generating products  
- 🧩 **Product Line Distribution**: Sales breakdown across categories like Classic Cars, Vintage Cars, Motorcycles, etc.  
- 📅 **Year-wise Trends**: Sales growth analysis from 2003 to 2005  
- 🎛 **Interactive Filters**: Year and Month slicers for dynamic analysis  

---

## 🛠 Tools & Technologies
- Power BI  
- Power Query  
- Excel  

---

## 📈 Key Insights
- Classic Cars contribute the highest share in total sales  
- Significant growth observed from 2003 to 2005  
- Certain products consistently outperform others in revenue generation  
- Sales distribution varies significantly across product lines
